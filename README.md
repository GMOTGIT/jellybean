# Jellybean

Jellybean is a static site generator created using Node.js.

# Getting Started

```diff
npm install --save jellybean
```
# Getting Help

```diff
jellybean --help
jellybean --h
```

# Version

```diff
jellybean --version
jellybean --v
```

# Running the Program

```diff
jellybean --input <file>
jellybean --input <folder>
jellybean --i <file>
jellybean --i <folder>
```

# Optional commands

```diff
jellybean --output <folder>
jellybean --stylesheet <URL>
```
